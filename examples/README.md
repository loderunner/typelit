# Examples

This directory contains practical examples demonstrating Typelit's type-safe template string functionality.

## Available Examples

- **[Email Templates](./email-templates/)** - Real-world email template examples showing nested paths, custom formatters, and complex data structures

## Running Examples

Each example directory includes its own README with specific instructions. Generally, you can run examples using:

```bash
npx tsx examples/<example-directory>/index.ts
```

## What You'll Learn

These examples demonstrate:

- Basic template creation with type-safe variables
- Nested context objects and path inference
- Custom variable creators with specialized formatting
- Real-world use cases like email templating
- Type safety in action with compile-time validation

Explore the examples to see how Typelit can help you create type-safe, maintainable template strings in your TypeScript projects.
